### 문제 설명

<p>대전과학고등학교의 학생 코이는 유명한 리듬게임 마이마이(maimai)를 즐기기 위해 단골 오락실 A를 찾았다.</p>

<p>오락실 A에는 $M$대의 마이마이 기기가 있으며, 각 기기마다 별도의 대기 카드판이 존재한다.</p>

<p>마이마이를 플레이하려면 자신의 대기 카드를 카드판에 놓아야 하며, 기기의 <strong>총 대기 시간</strong>은 코이가 도착했을 때 그 기기의 카드판에 이미 걸려 있는 모든 카드의 예상 플레이 시간의 합으로 계산된다.</p>

<p>오락실 A의 대기 시스템은 다음과 같이 작동한다.</p>

<ol>
<li>사람들은 도착한 순서대로 $M$개의 기기별 <strong>총 대기 시간</strong>이 가장 짧은 곳에 자신의 카드를 놓는다.</li>
<li>만약 <strong>총 대기 시간</strong>이 동일한 기기가 둘 이상이라면, 그중 아무 곳에나 카드를 놓는다.</li>
</ol>

<p>코이가 도착했을 때, 이미 $N$명의 사람들이 각자의 대기 카드를 걸어둔 상태였다.</p>

<p>이제 $N + 1$번째인 코이가 오락실 A에서 기다려야 하는 <strong>최소 대기 시간</strong>은 $M$개의 기기 중 <strong>총 대기 시간</strong>의 최솟값이다.</p>

<p>만약 코이가 오락실 A에서 기다려야 하는 <strong>최소 대기 시간</strong>이 다른 오락실 B까지 걸어가서 바로 플레이할 수 있는 시간 $K$초보다 길다면, 코이는 굳이 기다리지 않고 오락실 B로 이동해 대기 없이 곧바로 게임을 플레이할 것이다.</p>

<p>코이가 가장 현명한 선택을 할 수 있도록 오락실 A에서의 <strong>최소 대기 시간</strong>이 오락실 B로 가는 데 걸리는 시간보다 짧거나 같은지 판별해 보자.</p>

### 입력 

 <p>첫째 줄에 코이 이전의 대기 인원 $N$, 오락실 A의 기기 수 $M$, 오락실 B까지 가는 데 걸리는 정수 시간 $K$가 공백으로 구분되어 주어진다. $(1 \le N, M \le 1\,000;$ $1 \le K \le 10^9)$</p>

<p>둘째 줄에 사람들의 예상 플레이 시간을 나타내는 $N$개의 정수 $t_1, t_2, \cdots, t_N$이 도착한 순서대로 공백으로 구분되어 주어진다. $(1 \le t_i \le 10^6)$</p>

<p>모든 시간은 초 단위로 주어진다.</p>

### 출력 

 <p>코이가 오락실 B로 가는데 걸리는 시간과 비교해 오락실 A에서 기다리는 시간이 더 짧거나 같다면 <span style="color:#e74c3c;"><code>WAIT</code></span>을 출력한다. 그렇지 않다면 <span style="color:#e74c3c;"><code>GO</code></span>를 출력한다.</p>
